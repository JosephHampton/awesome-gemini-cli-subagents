# Roadmap

Planned directions for the collection. Contributions toward any of these are welcome.

## Near term
- Grow each category with more high-quality agents.
- Add usage examples showing how the main agent delegates to each subagent.
- A short "recipes" section pairing agents for common workflows (plan, build, review).

## Later
- Per-agent example prompts and sample outputs.
- Optional MCP server suggestions per agent where they add value.
- A script to install a single category rather than every agent at once.

Have an idea? Open a discussion or a pull request.
